## 📝 Important Point
---
- Membatalkan perubahan sebelum di commit
```cmd
git reset nama_file
```
- Membatalkan commit terkahir(tanpa menghapus perubahan)
```cmd
git reset --soft HEAD~1
```
- Membatalkan commit terakhir(menghapus perubahan)
```cmd
git reset --hard HEAD~1
```
